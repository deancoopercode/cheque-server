Requirements
================
Node v4.x needs to be installed prior to install node packages.

* please perform a npm install to ensure depentant modules are present


Execution
=========
To start the development server (service)
* node index.js

To run tests
* npm test


Procedure
================

1) Initialise a new package.json file.
* npm init

2) Install express.js and save it as a dependency
* npm install --save express

3) Install testing frameworks
* npm install mocha --save
* npm install chai --save
